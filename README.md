# Multiple Linear Regression: Multi-Channel Marketing ROI Analysis

## 📌 Project Overview
This project applies Ordinary Least Squares (OLS) Multiple Linear Regression to analyze marketing datasets, evaluate the return on investment (ROI) for different advertising channels, and provide empirical, data-driven budget allocation recommendations. This work was completed as part of the AI & Machine Learning Nextgen Cohort milestones.

## 🎯 Core Objectives
1. **Data Cleaning:** Handle missing values and standardize attributes within the multi-channel dataset.
2. **Multicollinearity Diagnostic:** Evaluate internal feature dependencies using a Correlation Matrix and Variance Inflation Factors (VIF).
3. **OLS Modeling:** Construct and fit a robust Multiple Linear Regression model using `statsmodels`.
4. **Assumption Validation:** Perform residual diagnostic analysis checking for Linearity, Normality, and Homoscedasticity.
5. **Business Optimization:** Translate quantitative coefficients into strategic corporate allocation plans.

## 📊 Key Statistical Findings

* **Model Accuracy (Adjusted R²):** `0.7353`  
  * *Interpretation:* The regression model successfully explains **73.5%** of the total variance observed in company Sales, indicating a highly reliable model for corporate planning.

* **Radio Advertising Budget:** * *Coefficient:* `8.2843` | *P-Value:* `8.84e-124`  
  * *Status:* **Highly Statistically Significant (p < 0.05).** * *Impact:* Holding all other variables constant, every 1-unit increase in Radio spend scales Sales by an average of **8.28 units**.

* **Social Media Advertising Budget:** * *Coefficient:* `0.1022` | *P-Value:* `0.9268`  
  * *Status:* **Not Statistically Significant (p > 0.05).** * *Impact:* Social Media performance fails to demonstrate a mathematically certain positive correlation with conversions when modeled alongside other core channels.

## 🚀 Strategic Budget Recommendations
Based on the empirical weights generated by the OLS model, the following business actions are recommended:
1. **Prioritize Radio:** Aggressively scale up capital allocation toward Radio advertising, as it represents the definitive growth driver.
2. **Audit Social Media Spend:** Immediately pause or restructure current Social Media budgets, transferring underperforming assets into high-converting traditional channels.
3. **Expand Feature Scope:** Future iterations should encode categorical metrics (like TV tiers) to test for multi-tiered conversion variance.
