# Simple Linear Regression – Marketing ROI Analysis

## Project Overview
This project analyzes a marketing dataset to evaluate the impact of different advertising channels (TV, Radio, Social Media) on Sales revenue. Using Python, `pandas`, and `statsmodels`, we built a Simple Linear Regression model to identify the channel that provides the highest Return on Investment (ROI).
