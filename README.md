# Feature Engineering – NBA Player Longevity Prediction

## 📌 Project Overview
This project establishes a robust feature engineering and data preprocessing pipeline using Python and `pandas` to transform raw NBA rookie performance metrics into an optimized dataset. The ultimate model target is `target_5yrs`, which predicts whether a player's professional career spans 5 or more years.
