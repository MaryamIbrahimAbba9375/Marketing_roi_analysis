# Simple Linear Regression - Marketing ROI Analysis

## 📌 Project Overview
This project applies Ordinary Least Squares (OLS) Simple Linear Regression to analyze a multi-channel marketing dataset, evaluate the return on investment (ROI) for different advertising channels, and provide empirical, data-driven budget allocation recommendations. This work was completed to fulfill the module milestones for the AI & Machine Learning Nextgen Cohort.

## 🎯 Core Objectives
1. **Data Preparation:** Load the dataset and systematically handle missing values.
2. **Exploratory Data Analysis (EDA):** Evaluate feature relationships using correlation matrices to isolate the strongest driver of Sales.
3. **OLS Regression Modeling:** Build and fit a statistically rigorous Simple Linear Regression framework using `statsmodels`.
4. **Assumption Validation:** Create and interpret diagnostic plots to verify classical linear regression assumptions (Linearity, Normality, and Homoscedasticity).
5. **Business Interpretation:** Translate quantitative model parameters into actionable stakeholder insights.
