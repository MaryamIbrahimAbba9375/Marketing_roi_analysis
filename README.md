# Logistic Regression – Airline Customer Satisfaction Analysis

## 📌 Project Overview
This project constructs a binomial Logistic Regression model using Scikit-Learn to predict whether a passenger will be "Satisfied" or "Neutral/Dissatisfied" based on airline customer service surveys. The insights are translated into target business investments to improve flight experiences and prevent customer churn.
