# LLM Serving Architecture Framework — FastAPI & Streamlit Production Gateway

This repository contains a decoupled, modular system architecture engineered to serve a local Large Language Model (TinyLlama-1.1B) via a fast asynchronous REST API backend, connected to a conversational Streamlit chat interface.
