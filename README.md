# K-Nearest Neighbor Classification – Pattern Recognition Mini Project

## 📌 Project Overview
This project constructs a K-Nearest Neighbors (KNN) pattern recognition engine using Python and Scikit-Learn to accurately classify samples from the Iris dataset. The model's hyperparameters are tuned via cross-validation, and features are standardized to protect geometric distance representations.
