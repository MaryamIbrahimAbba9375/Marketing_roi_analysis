# Logistic Regression – Airline Customer Satisfaction Prediction

## 📌 Project Overview
This project implements a Scikit-Learn Binomial Logistic Regression classification model to predict passenger satisfaction parameters based on historical survey metrics. This data allows the airline operation to maximize customer retention and improve resource allocation.
